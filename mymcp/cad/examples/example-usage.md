# CAD图像理解技能使用示例

## 示例1：简单零件分析

### 用户输入：
"我上传了一个机械零件的图片，请从CAD建模的角度分析这个图片，并生成可用于CAD建模的自然语言指令。"

### 技能响应：
```
# CAD图像理解报告

## 1. 图片基本信息
- 图片类型：3D模型截图
- 视图类型：等轴测视图
- 图片质量评估：清晰，视角良好

## 2. 识别的主要内容
- 主要物体：带孔矩形板
- 大致尺寸：200mm × 150mm × 20mm
- 关键特征：矩形基板、4个安装孔、倒角边缘

## 3. 详细几何分析
1. **矩形基板**：长200mm，宽150mm，厚20mm
2. **安装孔**：4个通孔，直径15mm
3. **孔位置**：距边缘25mm，对称分布
4. **倒角**：所有边缘有2mm倒角

## 4. 制造特征识别
1. **通孔**：4个安装通孔
2. **倒角**：边缘倒角特征
3. **对称特征**：孔对称分布

## 5. CAD建模自然语言指令
### 5.1 建模步骤概述
1. 创建矩形板草图并拉伸
2. 添加安装孔特征
3. 添加倒角特征

### 5.2 详细指令
#### 步骤1：创建矩形板
1. 在XY平面创建草图
2. 绘制200mm × 150mm的矩形
3. 约束矩形中心与坐标系原点对齐
4. 拉伸草图，厚度20mm

#### 步骤2：添加安装孔
1. 在矩形板上表面创建草图
2. 添加4个点，位置：距长边25mm，距短边25mm
3. 约束点对称分布
4. 创建直径15mm的通孔特征

#### 步骤3：添加倒角
1. 为矩形板所有边缘添加2mm倒角

### 5.3 关键尺寸和约束
- 矩形板尺寸：200mm × 150mm × 20mm
- 安装孔：直径15mm，位置对称
- 倒角：所有边缘2mm

## 6. 技术规范
- 尺寸公差：建议±0.5mm
- 表面粗糙度：建议Ra 3.2
- 材料要求：建议结构钢

## 7. 不确定性和假设
### 不确定性：
1. 精确尺寸基于图片比例估算
2. 倒角尺寸为视觉估算

### 假设：
1. 所有特征为对称设计
2. 安装孔为通孔
3. 材料为结构钢
```

## 示例3：装配体分析

### 用户输入：
"这是一个装配体的3D截图，请分析零件之间的配合关系。"

### 技能响应要点：
1. 识别主要零件：底座、支架、螺栓、垫圈
2. 分析配合关系：螺栓连接、面贴合、孔轴配合
3. 生成装配建模指令：按顺序添加零件、定义约束
4. 提供装配顺序建议

## 使用技巧

### 1. 提供高质量图片
- 确保图片清晰、光照均匀
- 多角度视图有助于更准确分析
- 如果有尺寸标注，确保清晰可读

### 2. 提供补充信息
- 说明零件的用途和功能
- 提供已知的关键尺寸
- 指定目标CAD软件

### 3. 验证分析结果
- 检查生成的指令是否合理
- 验证尺寸和特征的准确性
- 根据实际需求调整指令

### 4. 处理不确定性
- 对于不确定的信息，技能会明确标注
- 可以请求用户提供补充信息
- 基于保守估计生成指令

## 常见问题

### Q1：图片质量不佳怎么办？
A：技能会评估图片质量并告知限制。建议提供更清晰的图片或多角度视图。

### Q2：生成的指令可以直接使用吗？
A：生成的指令提供了建模思路和步骤，但可能需要根据具体CAD软件的功能进行调整。

### Q3：能处理复杂的自由曲面吗？
A：对于复杂的自由曲面，技能可能只能提供大致描述。建议提供多角度视图或使用专业扫描设备。

### Q4：如何提高分析准确性？
A：提供尺寸标注清晰的图纸、多角度视图、以及零件的用途信息可以显著提高分析准确性。

## 总结
CAD图像理解技能为机械设计、工程教育和产品开发提供了强大的辅助工具。通过将视觉信息转换为结构化的CAD建模指令，大大提高了从概念到3D模型的转换效率。
